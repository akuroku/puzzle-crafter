# Puzzlecrafter

### Lyhyesti

Ohjelma, jolla voi laatia ja ratkoa pulmia kuten Sudokuja sekä käsin että automaattisesti.

- Pit-raportti löytyy [täältä](https://htmlpreview.github.io/?https://github.com/akuroku/puzzle-crafter/blob/master/dokumentaatio/pit-raportti/index.html).
- Checkstyle-raportti löytyy [täältä](https://htmlpreview.github.io/?https://github.com/akuroku/puzzle-crafter/blob/master/dokumentaatio/checkstyle-raportti/checkstyle.html).
